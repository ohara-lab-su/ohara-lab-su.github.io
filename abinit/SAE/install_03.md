## はじめに

RMC形式への変換を始めとした各種ツール([マニュアル](README.md),[概要](http://www.spring8.or.jp/pdf/ja/ann_rep/14/054.pdf))を用います。

利促 VPN 上にある HPCS01 にはすでに導入していますので、その上での使い方を説明します。この環境を自力で導入する方法は[別途](install_01.md)記述してますのでそちらを算用してください。

- P1 (or POSCAR)で許されている基本並進ベクトルは全て変換できます。

## 環境変数のロード

```
source ~/env.sh
```

この作業で HPCS01 上で全て動くようになります。.bashrc などへの登録は避けてください(.basrhrc へ登録した状態でLSFで並列ジョブを投げるとジョブが正常終了しません）。面倒でも、使うときだけ、ログイン後に一度だけ実行してください。無論、この環境変数を登録時でも並列ジョブを投げるのはOKです。ログインシェルorシェルの自動登録するのだけは、NG です。

## 変換作業

1. VASP (P1) 形式を用意します。(VESTA で P1 形式、もしくはVASP形式で出力します。もしくは、同環境にある各種変換ツールで出力可能です）
2. Rvasp2cfg にて変換します。

ここでは、Li2MnO3.vasp というファイルを変換します。

```
Rvasp2cfg.py Li2MnO3.vasp
```

と実行すると。 Li2MnO3.vap.cfg が自動生成されます。この場合拡張子に特に意味は無いので。もし構造ファイルが POSCAR という名前でしたら

```
Rvasp2cfg.py POSCAR
```
と入力したら。POSCAR.cfg が出力ファイルになります。

### 第二引数を指定する(出力ファイルの指定)

は出力ファイル名を第二引数に指定します。

```
Rvasp2cfg.py Li2MnO3.vasp Li2MnO3
```

とすると、第二引数の Li2MnO3 に cfg を加えて。Li2MnO3.cfg が出力されます。
